# Tiles

*Max 8.5*

This Jitter Effect splits a texture into unique tiles and assigns them to a random position.<br/> 
The controls are:<br/>
**random**: set a new random position<br/>
**sorted**: reset tiles<br/>
**number of tiles**<br/>
**random offset** of the tiles<br/>
